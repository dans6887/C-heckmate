# C-heckmate
Get the status of your C: free space emailed to you. 


Checkmate is a simplet tool used to get the status of your C: freespace. 
Using gmail's smtp service you can run the script and get the report sent to any destination domain you like. 
This does require you to create an app specific password. 
You can create the app password for free using your google account here: https://myaccount.google.com/apppasswords. 
After entering your app password and email details you can run the script gnerate a simple free space alert.
